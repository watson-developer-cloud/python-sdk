watson_developer_cloud.authorization_v1 module¶

The v1 Authorization “service” that enables developers to retrieve a temporary access token

class AuthorizationV1(url='https://stream.watsonplatform.net/authorization/api', username=None, password=None, use_vcap_services=True)[source]¶

Bases: watson_developer_cloud.watson_service.WatsonService

Generates tokens, which can be used client-side to avoid exposing the service credentials. Tokens are valid for 1 hour and are sent using the X-Watson-Authorization-Token header.
